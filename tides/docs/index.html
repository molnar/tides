### Overview Needs Assessment is a learning support web application built with the [Google Angular Javascript framework](http://angularjs.org/). The JIRA project overview [can be viewed here.](http://feedback.csc.noaa.gov/browse/NEAS#selectedTab=com.atlassian.jira.plugin.system.project%3Asummary-panel) ************************************************************************************* ###Features + Built with the Angular framework from Google. + It has 12 awesome steps. + Case studies that exemplify the learning concepts on each step. + Questions found in each step can be downloaded in MSWord format. + Resources for continued learning. ************************************************************************************* ## Browser Compatibility + IE 9 and up + Chrome + Mobile Chrome + Firefox + Safari + Mobile Safari ************************************************************************************* ### Javascript Libraries This web app only uses the Angular framework **[Angular](http://angularjs.org/)** + [Developer docs](http://docs.angularjs.org/guide) + [API](http://docs.angularjs.org/api) + [Tuts](http://docs.angularjs.org/tutorial) ************************************************************************************* ### Hosting + **Production** -
